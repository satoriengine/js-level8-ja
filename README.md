# JavaScriptレベル8

プログラミングの学習を進める中で、動くコードを書くことには慣れてきたかもしれません。しかし、プログラムが複雑になるにつれ、「どこに処理を書くべきか」「どうすれば変更に強いコードになるか」という悩みが出てくるはずです。

このレベル8で学ぶのは、先人たちがたどり着いた設計の定石<span class="info">「デザインパターン」</span>です。

デザインパターンは、特定の場面で繰り返し発生する問題に対する<span class="info">「再利用可能な解決策」</span>です。

「クラス」という道具（ツール）を学んだ私たちが、次に手に入れるべきは、それらを<span class="info">どのように組み合わせて秩序あるシステムを築くか</span>という「設計思想」です。

このカリキュラムでは、モジュール化から生成の抽象化、振る舞いの差し替えに至るまで、実務で頻出する主要なパターンを習得し、コードの品質を一段引き上げる手法を学んでいきます。

## 内容
JavaScriptレベル8では以下のテーマを取り扱います。

* デザインパターンとは
* モジュールパターンとカプセル化
* ファクトリパターン
* シングルトンパターン
* オブザーバーパターン
* コマンドパターン
* デコレーターパターン
* テンプレートメソッドパターン
* ストラテジーパターンとコンポジション

「ECMAScript2015」以降に準拠していますが、本カリキュラムでは上記の分野のみを取り扱います。

## 対象
JavaScriptレベル8は以下のような方を対象にしています。

* JavaScriptの学習が初めての方
